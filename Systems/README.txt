#### Folder INFO ####

Each sub-folders contain system related input files to start WE simulations


A) WE simulations were started from 'folded.gro' structures.

B) 'unfolded.gro' files were used as the target conformations.

C) 'topol.top' is the corresponding topology file.

D) 'eigenvectors.txt' file contains the TICA eigenvectors.

E) 'final_data.txt' contains MFPT values that we reported.
